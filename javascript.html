Stage Two: Tracking Multiple Items with Arrays

What is an Array

An array is what's known as a data structure: a way to store and organize information. Example of an array: var someList = ['carrots', 30, 'milk'];

Accessing Items in an Array

To access a particular item value in an array use the corresponding index value: console.log(someList[0]); will return the first item 'carrots' in the someList array. The return value does not print the quotes to the screen, just the letters enclosed in quotes. We start counting with 0 to represent the first position.

If we try to access a item value that is not in the list, the console will return 'undefined'. Example: someList[20] will return undefined in the console if there are only 20 items in the list. 19 represents the 20th item, so 20 would represent an undefined 21st item.

Adding Data to Arrays

Using the push method allows us to add ones or more items to the end of an array. Example syntax: nameOfArray.push('apple', 'butter', 'cheese');

someList.push('apple', 'butter', 'cheese'); will add the string 'apple' to the fourth position 'butter' to the fifth, and 'cheese' to the sixth position in the someList array.

Unshift works likes the push method, but it adds items to the beginning of the array.

Removing Items from Arrays

The pop method removes the last item in the array list and returns that value. If you want to hold that value, you can declare a var and set it equal to the arrayListName.pop();

The shift method will remove the first item in the list and return its value.

Handy shortcut for the console: Press the up arrow will cycle thru the previous commands beginning with the last one used, going backward.

Using For Loops with Arrays

We can use the 'for' loop and the 'length' property for the condition to cycle thru students array and print to the screen. We use the length property to accurately determine how many items are currently in the array. This allows us to use that property in the condition so that we run the loop the required number of times.

We can hold the value of the array in a variable, and use the join method to print out the list seprated by the characters of our choosing. The last item will not be seperated by any characters.

We can use the concat method to combine two arrays; the first array in the syntax will be printed first followed by the array called inside the parentheses. Example: var currentStudents.concat(newStudents);

It is wise to write short bits of code and then test. This method of coding makes it easier to spot errors.

Stage Three: Tracking Data Using Objects

The Object Literal

An object has properties and methods; properties are variables belonging to the object and methods are somthing that can be done to the object.

To create an object, use this syntax: var objectName = { key: "Value", key: [89, 85, 40] }; The word 'key' represents the property/value pair. You can name it as you would a normal variable; don't start with a number, and use lowercase, uppercase, $, _ and numbers. It is good to use adequate white space and line spaces when formatting the object to make it easier to read.

Below is an example of a properly formatted object. ‘name:’ is the 'key', and "Dave" is the 'key value'. ‘grades:’ is the 'key', and [80, 85, 90, 95] is the 'key value'.

Accessing Object Properties

Example object: var person = { name: "Dave" }; You may use brackets: alert(person['name']); or dot notation: alert(person.name); to access an object.

To set a key property/value pair, simply type the object variable-name, and use dot notation using the key-name along with an equals sign. Example: person.name = "Matt";

You can also add new properties to an object by adding a new key name by using dot notation. Example: person.country = "Bolivia";

We can use methods of objects by using the period between the object and method. Example: person.skills.join(', '); This will take the value of the property of skills, which is an array in this example--since it is an array, we can use the .join method(', ') to print the list of skills with a comma and a space between each one, save the last skill.

Also, if we want to target a specific skill, we can use its index number. console.log(person.skills[2]); would return the 3rd item in the skills array.

Using 'for in" to Loop Through an Object's Properties

Syntax of a 'for in' loop: for ( var keyProperty in student) { programming statements; } Where student would be the object and keyProperty is the variable that will hold the looped key property names. The key properties will be returned. Below is the formatted example.

To access the key property values, we use bracket notation. Example: for (var keyProperty in student) { console.log(student[keyProperty]); } Below is a formatted example syntax that returns the looped values of the keyProperties in the loop.

We must use bracket notation to call a property value because dot notation (example: student.keyProperty) will cause the interpreter to look for a property named labelled 'keyProperty'. So it will return undefined. If we named our keyProperty variable 'keyProperty' then the interpreter will return all of the property names if we request them like this: for (keyProperty in student) {console.log(keyProperty)}.

If we want the interpreter to return the keyProperty names AND values seperated by a colon, we can type this: console.log(keyProperty, ': ', student[keyProperty]); Below is an example of the formatted syntax.

JSON is a string that contains keyProperties and value pairs. JSON can be formatted to appear as an Object for ease of human understanding. It allows servers to 'talk' to each other.
